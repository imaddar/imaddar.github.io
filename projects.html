RL for Crowded Navigation

We extended a POMDP planner with deep learning policies so an actor can move through crowded environments with lower performance overhead compared to existing solutions. By leveraging trained policy and value networks, we applied the work of the BetaZero paper to our crowded environment. This project was an extension of the work done by our colleagues Gupta et al.

  • Julia
  • Flux.jl
  • Reinforcement Learning
Sherlock Sources

For a hackathon we built a chrome extension to help verify claims found on the internet. By highlighting a chunk of text, we clean the text, optimize it for search query, and scrape websites to compare with top results. By doing the heavy lifting beforehand, we can leverage much smaller, local models to speed up inference and reduce costs while maintaining exceptional performance.

  • Javascript
  • Python
Soccer Tracker

Wrote a python script to ingest clips from soccer matches to track players. By finetuning a YOLOv8 model, it can accurately detect players, referees, and balls. Futhermore, by using k-means clustering to determine player teams, useful statistics can also be extracted like percentage of possesion, player speeds, and individual player numbers.

  • Python
  • YOLO
